# ValidarRuc

Hola.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'validar_ruc', :git => "git://github.com/ronaldmacedo/validar_ruc.git"
```

And then execute:

    $ bundle

## Usage

Invocar directamente:

ValidarRuc.validar(20000000001)
